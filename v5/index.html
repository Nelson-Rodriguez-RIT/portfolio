About Me

Who am I?

My name is Nelson Rodriguez and I'm currently a second-year Game Design and Development student at the Golisano College of Computing and Information Sciences.

When it comes down to it, my passion is software programming, especially in the context of games and interactive media. I truly enjoy learning about the foundational systems (i.e. physics engine, graphical pipelines, gameplay systems) behind many of the awesome games that I play and make.


What can I do?

My primary skillset revolves around programming, however, I do have expierence using asset creation tools (Maya, Shotcut, Paint.Net, and Aseprite).

While taught both C# and HTML/CSS/JavaScript from college classes, I have gone out and expiremented with C++, OpenGL (and some of the shader programming that comes with it), Lua, Love2D, and even some networking (using Socket.io).

Taiko No Tatsujin Web Recreation

Taiko No Tatsujin (lit. 太鼓の達人) is a Japanese rhythm game revolving around pressing specific keys to hit colored "notes", and is timed to the beat of the music.

This web recreation contains a main menu, a settings menu, a tutorial, a song selection, and the gameplay itself. The "beatmaps" you play are user-generated and supplied by the popular rhythm game "Osu!", allowing for nearly limitless replayability.

Done purely in HTML, CSS, and JavaScript with NO canvas element. A single external library, named JSZip, is used for extracting user-generated content. (Note: Textures belong to Bandai Namco; I did not make any of these assets)

Real-Time Jello Chat

A web application that allows for creating and joining custom user-created lobbies, with Discord-styled messaging inside of each lobby. All of these are updated in real-time via a custom API used to help communicate between browsers through a personally hosted dedicated server.

Lobbies can be password protected and users, once connected to a lobby, can message and send GIFs. GIFs are provided via the Tenor API , with users allowed to search with keywords for specific GIFs they may want. Additionally, users can upload their own profile pictures (which are stored locally, alongside information such as their username, for convenience).
